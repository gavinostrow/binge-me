# Binge Me

## Overview
Binge Me is a project designed to help users keep track of their favorite shows and movies. Users can search for titles, create watchlists, and receive recommendations based on their viewing history.

## Project Setup Instructions

### Prerequisites
- Node.js (version 14 or above)  
- NPM (or Yarn)

### Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/gavinostrow/binge-me.git
   ```
2. Navigate into the project directory:
   ```bash
   cd binge-me
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```

### Usage
After starting the application, navigate to `http://localhost:3000` in your web browser to access the app!
